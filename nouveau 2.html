Depuis ses débuts en 1951 avec le langage FORTRAN (Formula Translation),
conçu pour les calculs scientifiques, l'histoire des langages de programmation a été marquée par diverses
évolutions. Le COBOL (Common Business-Oriented Language) en 1959 a révolutionné la programmation
commerciale, tandis que le LISP (LISt Processing) en 1958 a ouvert la voie à la programmation fonctionnelle.
Dans les années 70, le langage C a émergé, offrant une efficacité et une portabilité accrues.
Les années 80 ont vu l'avènement du C++, qui a introduit la programmation orientée objet.
En 1991, Python est apparu avec sa syntaxe claire et expressive, mettant l'accent sur
la lisibilité du code et la productivité du développeur, marquant ainsi
le début d'une ère de langages de haut niveau et d'une communauté de développement florissante.

Les systèmes d'exploitations:

Unix (1969) - Conçu à l'origine par AT&T Bell Labs, Unix est un système d'exploitation multi-utilisateur et multitâche, largement utilisé dans les environnements serveur et de superordinateurs.


MS-DOS (1981) - Développé par Microsoft, MS-DOS était le système d'exploitation dominant pour les ordinateurs personnels IBM PC et compatibles dans les années 1980.


Mac OS (1984) - Développé par Apple, Mac OS est le système d'exploitation utilisé sur les ordinateurs Macintosh. Il a évolué au fil des ans, passant par plusieurs versions majeures telles que Mac OS X et macOS.


Microsoft Windows (1985) - Lancé par Microsoft, Windows est devenu le système d'exploitation le plus largement utilisé sur les ordinateurs personnels, offrant une interface graphique conviviale et une large gamme de fonctionnalités.


Linux (1991) - Créé par Linus Torvalds, Linux est un système d'exploitation open source basé sur Unix, largement utilisé dans les environnements serveur, les appareils embarqués et les systèmes informatiques de bureau.


iOS (2007) - Développé par Apple, iOS est le système d'exploitation mobile utilisé sur les appareils iPhone, iPad et iPod Touch, offrant une expérience utilisateur optimisée pour les appareils mobiles.


Android (2008) - Initié par Android Inc. puis acquis par Google, Android est un système d'exploitation mobile open source basé sur Linux, utilisé par de nombreux fabricants de smartphones et tablettes.


Windows Phone / Windows Mobile (2010) - Microsoft a proposé ces systèmes d'exploitation mobiles pour les smartphones, offrant une intégration étroite avec les services Microsoft tels que Office et Outlook.


Chrome OS (2011) - Développé par Google, Chrome OS est un système d'exploitation basé sur le navigateur Chrome, conçu principalement pour les appareils Chromebook, offrant une expérience de navigation web centrée.


Ubuntu (2004) - Ubuntu est une distribution Linux populaire basée sur Debian, reconnue pour sa convivialité et sa facilité d'utilisation, ciblant à la fois les utilisateurs débutants et avancés.


Ces systèmes d'exploitation ont chacun leurs propres caractéristiques et sont utilisés dans une variété de contextes, des ordinateurs personnels aux appareils mobiles et aux serveurs.
